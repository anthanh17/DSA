Array include:

- Static array
  - Fixed size array
- Dynamic array
  - resizeable array
  - Python: list - python chỉ có mảng động

Đặc điểm:

- data trong mảng được lưu trữ liên tiếp nhau trên memory
- random access: O(1)
- thêm và xóa phần tử ở cuối mảng (chỉ mảng động): O(1)
- thêm và xóa 1 phần tử bất kì trong mảng: O(n)
